# 🤖 Телеграм-бот для бронирования квартир VDKcity

Бот для поиска и посуточного бронирования квартир во Владивостоке. Позволяет пользователям просматривать доступные объекты по районам, выбирать даты проживания через календарь, оставлять заявки и отслеживать историю своих запросов.

Проект интегрирован с сайтом-лендингом на платформе Tilda (`vdkcity.ru`).

---

## ✨ Возможности

- **🏠 Просмотр квартир:** Удобная навигация по районам города с помощью Reply и Inline кнопок.
- **📅 Выбор дат:** Выбор дат заезда и выезда через интерактивный календарь.
- **📝 Пошаговое бронирование:** Сбор необходимых данных (имя, телефон, количество гостей) в формате удобного диалога.
- **🔔 Уведомление администратора:** Моментальная отправка новых заявок владельцу.
- **📂 История для пользователей:** Возможность посмотреть свои прошлые запросы и их статус.

---

## 🛠 Технологии и стек

- **Язык программирования:** Python 3.14
- **Фреймворк:** `python-telegram-bot` v21.10
- **ORM:** Peewee
- **База данных:** SQLite (локальный файл `bookings.db`)
- **Конфигурация:** `python-dotenv`

---

## 📋 Команды бота

### 👤 Пользовательские команды
- `/start` — Запустить главное меню бота.
- `/help` — Показать справочную информацию.
- `/history` — Показать историю ваших заявок.

### 👑 Административные команды (только для `ADMIN_ID`)
- `/bookings` — Вывести список всех заявок.
- `/stats` — Показать общую статистику бронирований.
- `/find` — Найти конкретную заявку по её ID.
